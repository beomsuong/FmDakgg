#사용 패키지

1.  flutter_screenutil -> 기기별 해상도 맞춤 적용
2.  go_router -> 페이지 라우팅
3.  riverpod -> 상태관리자

flutter_screenutil를 회사에서 적용 여부를 테스트 하기위해 비효율적이라도 비율을 사용한 위젯 선언 대신 flutter_screenutil를 사용하여 직접 크기를 입력해 위젯 디자인 중
